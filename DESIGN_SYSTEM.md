# Design System

## Product Name

BiteFlow

## Domain

Food delivery mobile app

## Design Goals

- Make food discovery feel quick and visually rich.
- Keep the checkout path short and obvious.
- Use motion to confirm user actions without slowing the flow.
- Present a realistic mobile UI suitable for a UI/UX assignment or portfolio project.

## Color Palette

| Token | Hex | Usage |
|---|---|---|
| Ink | #202326 | Primary text and main CTA |
| Coral | #E9583F | Add buttons, cart badge, brand accent |
| Teal | #227C77 | Secondary accent and offer banner |
| Mint | #DFF3DF | Success confirmation background |
| Gold | #F4BD45 | Food imagery and rating accent |
| Paper | #FFFAF1 | App background |
| Line | #E8E1D5 | Borders and dividers |

## Typography

Recommended fonts:

- Inter
- SF Pro
- Segoe UI fallback

Hierarchy:

- Hero heading: 31 px, bold
- Section heading: 18 px, bold
- Card title: 14 px, bold
- Metadata: 11-13 px, bold, muted color
- Buttons: bold labels with compact sizing

## Layout

Target frame:

- 390 x 844 px mobile frame

Spacing:

- Main screen padding: 18 px
- Card grid gap: 12 px
- Bottom navigation height: 74 px
- Cards and controls use 8 px radius
